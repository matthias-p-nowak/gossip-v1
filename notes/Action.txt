====== Action ======
Created Sunday 13 September 2020

It's an interface with several implementations.



There is a current action in the tester and a slice of optional actions.
Usually the actions take a message from the input channel, but the main action is in Execute. This is for optional messages, which must be tried with a given message. 

===== Delay Action =====
uses time.After to get a notification when that duration is gone

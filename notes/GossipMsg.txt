====== GossipMsg ======

Datastructure that represent a SIP message. 

===== GossipMsgHeaders =====
Headers in SIP are basically a header name and a header value. However, header names can appear several times like in VIA, Route, and RecordRoute headers. Hence, each header is a list of values.
